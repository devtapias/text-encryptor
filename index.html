Para encriptar:

  1. 1. Escribe o pega el texto que deseas encriptar.
  2. 2. Preciona en el botón "Encrypt".
  3. 3. El mensaje encriptado estará disponible en la parte izquierda.
  4. 4. Usa el botón "Copy" para que puedas copiar el texto y enviarlo a quien corresponda.

Para Desencriptar:

  1. 1. Copia el mensaje que te enviaron encriptado.
  2. 2. Usa el botón "Paste Text" para pegar el mensaje.
  3. 3. Preciona en el botón "Decrypt".
  4. 4. El mensaje desencriptado estará disponible en la parte izquierda para que puedas leerlo.

Usa el botón "New" para volver a empezar.

🕴Only lowercase letters and no accents.
